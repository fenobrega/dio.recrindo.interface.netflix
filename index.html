Sobre

Recriando interface do principal site de streaming, com Felipe Aguiar. Neste projeto aprendi como estruturar um layout, técnicas de CSS3 com containers e variáveis, como posicionar elementos com Flexbox e como utilizar plugins JQuery. Felipe Aguiar,S propôs que fizéssemos uma nova roupagem e déssemos vidas aos botões.
Decidi fazer um streaming de adoção de animais e acrescentei um form de contatos para treinar meus conhecimentos. Os botões vão levar para o site da Luisa Mell "Instituto Luisa Mell", onde contará história de cada cachorro adicionado no site.
